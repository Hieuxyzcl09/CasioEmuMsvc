# CasioEmuMsvc

An emulator for nX-U16/100 MCU (and nX-U8/100 MCU as well).

With debugger and built-in disassembler.

# Usage

* Download Visual Studio 2022 (or Build Tools)

* Clone this repo

* Build with VS or using `msbuild`
